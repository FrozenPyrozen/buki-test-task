## Instaliation

1. Clone this repository:
```
$ git clone https://github.com/FrozenPyrozen/buki-test-task.git
```
2. Change the current directory to **buki-test-task**:
```
$ cd ./buki-test-task
```
3. Install dependencies using npm/yarn:
```
$ npm install
```
```
$ yarn install
```
4. Run app in terminal:
```
$ npm start
```
